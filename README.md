## About The Project

![Demo](/public/demo.png)

A Blog Web based on https://jsonplaceholder.typicode.com/. Made with React

### Live Demo

[Bloggg.](https://mybloggg.web.app/)

### How to Run

- `npm install`
- `npm start`

### Built With

- [TailwindCSS](https://tailwindcss.com/)
